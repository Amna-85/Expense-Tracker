Expense Tracker:
A comprehensive Spring Boot web application for personal expense management with analytics and budgeting features.

 Features:
User Registration & Authentication - Secure login system

Expense Management - Add, view, and categorize expenses

Analytics Dashboard - Visual spending insights with charts

Budget Tracking - Set and monitor monthly spending limits

Category-wise Analysis - Breakdown of expenses by category

 Tech Stack:
Backend: Spring Boot 3.x, Java 17+

Frontend: Thymeleaf, Bootstrap 5, Chart.js

Data Storage: JSON files (Jackson)

Authentication: Session-based

Build Tool: Maven
